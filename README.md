# logistic-regresion
Objective
Build a binary classifier using logistic regression.
Dataset
Used a simple CSV dataset with 'diagnosis' as a feature to classify a binary target.
 Tools
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Visual Studio
Metrics Evaluated
- Confusion Matrix
- Precision, Recall
- ROC-AUC
- Sigmoid function used via Logistic Regression
